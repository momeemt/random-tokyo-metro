# ランダム東京メトロ

東京メトロの駅をランダムに選んで、知らない街を冒険しよう！

https://tokyometro.momee.mt

## 概要

「ランダム東京メトロ」は、東京メトロ9路線・全180駅からランダムに駅を選んでくれるWebアプリです。次に降りる駅を運任せにすることで、普段は行かない街や知らないお店との出会いを楽しめます。

## 遊び方

### 基本の流れ

1. **出発駅を選ぶ** - 路線を選択し、今いる駅をタップ
2. **ゲームを開始** - ボタンを押すとランダムに目的地が決定！
3. **目的地へ向かう** - 表示された経路で移動（Yahoo!乗換案内へのリンクあり）
4. **到着したら次へ** - 「次の駅を選ぶ」で新しい目的地へ
5. **好きなだけ続ける** - 疲れたら「ゲーム終了」

### 画面の見方

- 経路は路線カラーで視覚的に表示
- 乗り換え駅には「乗換」バッジ
- 何回目の目的地かが一目でわかる
- ゲーム履歴はタブで切り替え可能

## こんな時に使えます

### 休日の街歩きに
行き先を決めずに出かけたい休日に。ランダムに選ばれた駅で降りて、周辺を散策してみましょう。思いがけない発見があるかもしれません。

### 知らないお店の開拓に
「いつも同じお店ばかり...」そんな時は運任せで新しい街へ。降りた駅の周辺で気になるお店を探してみては？

### 友達との冒険に
「どこ行く？」「どこでもいいよ」の無限ループを解決。ランダムに決まった駅なら、誰のせいでもありません。

### 東京メトロ全駅制覇チャレンジ
履歴機能で訪れた駅を記録できるので、全180駅制覇を目指すのも面白いかも？
